# 古诗卡片分享

将古诗词排版成精美卡片，导出为图片分享到小红书等社交平台。

## 功能特性

- **三栏布局**：左侧输入、中间预览、右侧设置
- **多种卡片比例**：3:4 (600×800)、9:16 (540×960)、1:1 (720×720)
- **背景支持**：内置背景图 + 本地上传自定义背景
- **丰富排版**：字号、行间距、内间距、文字颜色、字体选择
- **文本格式**：标题(H1/H2/H3)、高亮、引用、有序/无序列表
- **预览缩放**：+/- 按钮调整预览比例
- **导出图片**：导出 PNG 格式，可直接分享到小红书

## 快速开始

```bash
npm install
npm run dev
```

访问 http://localhost:5174/

## 使用方法

### 输入文案

在左侧输入区输入文案，支持以下格式：

| 格式 | 语法 | 示例 |
|------|------|------|
| 一级标题 | `# 标题` | `# 静夜思` |
| 二级标题 | `## 标题` | `## 李白` |
| 三级标题 | `### 标题` | `### 唐诗` |
| 高亮 | `**文字**` | `**明月光**` |
| 引用 | `> 文字` | `> 疑是地上霜` |
| 无序列表 | `- 内容` | `- 床前明月光` |
| 有序列表 | `1. 内容` | `1. 床前明月光` |

### 快捷按钮

输入区上方提供快捷按钮，点击可快速插入格式标记。支持多行选中后批量添加列表格式。

### 卡片设置

右侧设置区可调整：

- **比例**：选择卡片尺寸比例
- **背景**：选择内置背景或上传自定义图片
- **背景色**：纯色背景颜色
- **字号**：12-48px
- **行间距**：1-3倍
- **内间距**：横向/垂直 10-100px
- **文字颜色**：自定义颜色
- **字体**：5款古风字体 + 系统字体
- **对齐**：文本水平/垂直对齐
- **版权信息**：固定在卡片底部
- **引用色块**：开关引用左侧色块
- **标题间距**：标题下方间距
- **正文间距**：正文上下间距

### 预览与导出

- 中间区域实时预览卡片效果
- 右上角 +/- 按钮调整预览缩放比例
- 鼠标悬停卡片显示"导出"按钮
- 点击导出保存 PNG 图片

## 内置字体

| 字体 | 风格 |
|------|------|
| 汇文明朝体 | 古典正式 |
| 寒蝉行楷体 | 行书风格 |
| 阿里妈妈刀隶体 | 隶书风格 |
| 汉仪中隶书简 | 隶书简体 |
| 京华老宋体 | 宋体风格 |

## 技术栈

- Vite + React + TypeScript
- Tailwind CSS
- html2canvas (导出图片)

## 项目结构

```
src/
├── components/
│   ├── Card/           # 卡片渲染组件
│   ├── InputArea/      # 输入区域
│   ├── PreviewArea/    # 预览区域
│   └── SettingsPanel/  # 设置面板
├── types/              # TypeScript 类型
├── utils/              # 工具函数
│   ├── backgrounds.ts  # 背景图配置
│   ├── export.ts       # 图片导出
│   ├── fontManager.ts  # 字体管理
│   └── textSplitter.ts # 文本拆分
└── App.tsx
```
